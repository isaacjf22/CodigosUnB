## 📂 Conteúdo do Repositório

Este repositório reúne exercícios e projetos de fixação organizados por tema:

* 🔹 **Lógica de Programação em C** — exercícios de fixação com loops, condicionais, vetores, ponteiros e manipulação de memória.
* 🔹 **Lógica de Programação em C#** — exercícios de fixação com loops, condicionais, arrays, strings e manipulação de dados.
* 🔹 **Programação Orientada a Objetos (C#)** — encapsulamento, construtores, herança, polimorfismo, properties, entidades e services.
* 🔹 **Lógica de Programação em Python** — exercícios de fixação com estruturas de dados, funções, manipulação de strings e arquivos.
